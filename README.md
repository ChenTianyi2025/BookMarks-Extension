# 书签管理器 (Bookmarks Manager)

一个简单易用的浏览器书签管理插件，支持添加、删除、搜索和快速访问书签。

## 功能特性

- 📚 **书签管理** - 添加、删除和查看书签
- 🔍 **搜索功能** - 快速搜索书签标题、网址和描述
- �️ **分组管理** - 为书签创建和分配分组，支持多分组
- 📤 **导出功能** - 将书签导出为JSON文件
- 📥 **导入功能** - 从JSON文件导入书签
- � **当前页面支持** - 一键添加当前浏览页面为书签
- 💾 **数据存储** - 使用浏览器存储同步保存书签数据
- 🎨 **简洁界面** - 现代化的用户界面设计


## 安装方法

### 开发者模式加载（开发测试）

1. 打开 Chrome 浏览器，进入扩展程序页面：
   - 在地址栏输入 `chrome://extensions/`
   - 或者点击菜单 → 更多工具 → 扩展程序

2. 开启"开发者模式"（右上角开关）

3. 点击"加载已解压的扩展程序"

4. 选择本项目所在的文件夹（包含 `manifest.json` 的文件夹）

5. 插件将成功安装并显示在扩展程序栏中


## 使用方法

### 基本操作

1. **打开插件**：点击浏览器工具栏中的书签管理器图标
2. **查看书签**：主界面显示所有已保存的书签
3. **搜索书签**：在搜索框中输入关键词进行过滤
4. **访问书签**：双击任意书签即可在新标签页中打开

### 分组筛选

1. 在主界面的"分组筛选"区域
2. 点击任意分组按钮进行筛选（可多选）
3. 点击"显示全部"按钮清除筛选

### 导出书签

1. 点击"导出书签"按钮
2. 浏览器会自动下载JSON文件
3. 文件名格式：`bookmarks_YYYY-MM-DD.json`

### 导入书签

1. 点击"导入书签"按钮
2. 选择要导入的JSON文件
3. 确认导入操作
4. 选择导入模式：
   - **合并模式**：保留现有书签，导入新书签（根据URL去重）
   - **替换模式**：完全替换现有书签

### 添加书签

1. 点击"添加书签"按钮
2. 在弹出的表单中填写：
   - **标题**：书签名称（必填）
   - **网址**：完整的URL地址（必填）
   - **描述**：书签描述（可选）
3. 选择或创建分组（可选）：
   - 勾选现有分组（可多选）
   - 输入新分组名称并点击"添加分组"
4. 点击"使用当前页面地址"可快速获取当前页面URL
5. 点击"保存"完成添加

### 删除书签

1. 点击"删除书签"按钮
2. 从下拉列表中选择要删除的书签
3. 点击"确认删除"完成操作

## 技术实现

### 核心技术
- **Manifest V3**：使用最新的浏览器扩展规范
- **Chrome Storage API**：用于数据持久化存储
- **Popup Windows**：独立的弹窗界面设计
- **Message Passing**：组件间通信机制

### 主要文件说明

- **manifest.json**：插件配置文件，定义权限和资源
- **background.js**：后台服务，处理数据存储和消息通信
- **popup.html/js**：主界面，显示和搜索书签
- **popup_form.html/js**：表单界面，处理添加/删除操作
- **utils.js**：工具函数，包含URL验证、过滤等功能
- **styles.css**：统一样式文件

## 权限说明

插件需要以下权限：
- `storage`：存储书签数据
- `activeTab`：获取当前活动标签页信息
- `tabs`：管理浏览器标签页


## 注意事项

- 书签数据存储在浏览器的同步存储中，会在不同设备间同步（如果已登录Chrome账号）
- 插件仅支持HTTP/HTTPS协议的URL
- 删除书签操作不可撤销，请谨慎操作
- 导入书签时会自动验证数据格式
- 分组筛选支持多选，可以同时查看多个分组的书签
- 导入导出的JSON文件包含分组信息
